# Food-Tracker
 ### 1. 프로젝트명
           * Swift를 사용한 나만의 맛집 추천 메뉴 리스트 애플리케이션 
  
 ### 2. 프로그램 설명 
           * TableViewController와 ImagePickerController를 사용해 메뉴 UI를 구성  
           * 카메라 롤에서 선택한 이미지를 이미지 뷰에 출력
           * 리스트 추가, 삭제 기능을 통해 메뉴 리스트를 구현
 
 ### 3. 사용 언어
           * Swift
